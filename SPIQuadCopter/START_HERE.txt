================================================================================
                        INSTALLATION SUMMARY
================================================================================

Minimum Installation (5 minutes):

  1. Install Python 3: apt-get install python3 python3-pip
  2. Install apio: pip install apio
  3. Install Gowin tools: apio install gowin
     (This takes 10-15 minutes for tool download)

Verify:
  apio system-info

That's it! You're ready to build.

================================================================================
                         COMMON COMMANDS
================================================================================

Build:
  apio build

Program:
  apio upload

Test SPI Slave:
  cd spiSlave && make simulate && make wave

Test LED Blinker:
  cd src && make simulate && make wave

Automated (recommended):
  ./build.sh

Clean:
  apio clean
  make clean
